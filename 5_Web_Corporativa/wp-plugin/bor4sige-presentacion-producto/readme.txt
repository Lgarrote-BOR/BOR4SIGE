Publica en WordPress, idéntica y a pantalla completa, la web de presentación del producto BOR4SIGE (bilingüe ES/EN).

== Description ==

Empaqueta la presentación del producto BOR4SIGE (Sistema Integrado de Gestión Empresarial) y la
publica dentro de WordPress sin perder fidelidad visual. Diseño bilingüe español/inglés con
conmutador de idioma.

Al activarlo:
1. Crea automáticamente una página publicada con el slug `/bor4sige/`.
2. Esa página sirve la presentación A PANTALLA COMPLETA, sin que el tema altere el diseño.

Shortcode:
* `[bor4sige_presentacion height="100vh"]` — incrusta la presentación mediante un iframe aislado.

Añade `?embed=1` a la URL de la página para verla CON la plantilla del tema.

== Installation ==

1. WordPress: Plugins → Añadir nuevo → Subir plugin.
2. Selecciona `bor4sige_web_presentacion.zip` → Instalar ahora.
3. Activa "BOR4SIGE · Presentación de Producto".
4. Visita `https://TU-DOMINIO/bor4sige/`.
